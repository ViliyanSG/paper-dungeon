# 📜 Paper Dungeon

Дигитална версия на pen-and-paper дъндживон кроулър, направена с **Godot 4**.
Едно приложение → Windows, Linux и (скоро) Android.

## Core механики (Фаза 0)
- Grid движение чрез чертане на линия с „молив".
- Рандомизатор „Завъртане на молива" (заместник на зара).
- Бой при пресичане с враг, HP система.
- Капани, сандъци, мини-бос, изход.

## Стартиране (за разработка)
1. Свали [Godot 4.3](https://godotengine.org/download).
2. Отвори `project.godot` с редактора.
3. Натисни ▶ (F5).

## Автоматични билдове
При всеки push към `main`, GitHub Actions експортва Windows + Linux билдове
и ги качва в **Releases**. Свали готовия `.exe` оттам — не ти трябва Godot за да играеш.
